3.1.4 Geospatial Data

  • defined by exact location on the earth surface
  • information attached to coordinates
set.seed(10)
mof_trees = st_read("data/mof/mof_trees.gpkg") %>% 
    na.omit() %>%
    slice_sample(n = 5) %>% 
    select(-socialposition) %>% 
    mutate(dbh = round(dbh, 2))
